# Package: src.sentrymode

1. Status: Application composition package for the SentryMode runtime.
2. Logic: Exposes CLI entrypoint, shared data adapter module, factor registry package, and monitoring runtime package.
3. Constraints:
   - Keep CLI wiring in `__main__.py`; avoid embedding argument parsing into factor modules.
   - Cross-package settings access should flow through `monitoring.settings.Settings`.
   - Runtime imports should remain inside `sentrymode` package boundaries (no `tests` or `docs` imports).

## Member List

- `__init__.py`: package metadata (`__version__`).
- `__main__.py`: CLI parser and command dispatch.
- `market_data.py`: provider protocol plus Yahoo/Glassnode implementations for normalized daily series.
- `factors/`: pluggable factor implementations and registry.
- `monitoring/`: domain models, settings, notifier, and runner orchestration.
