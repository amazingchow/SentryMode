# Package: src.sentrymode.factors

1. Status: Factor plugin package for market-signal evaluation logic.
2. Logic: Hosts concrete factor implementations and registry helpers that instantiate factors for the runner.
3. Constraints:
   - Factors should emit `FactorResult` and follow `Factor` protocol semantics from `sentrymode.monitoring`.
   - Shared market data loading should use `sentrymode.market_data` seams rather than custom ad-hoc HTTP code when possible.
   - Registry updates in `__init__.py` are required whenever a factor module is added or removed.

## Member List

- `__init__.py`: factor registry and factory helpers.
- `ahr999.py`: BTC valuation factor (AHR999) and single-factor runner helpers.
- `vix.py`: VIX risk-light factor and single-factor runner helpers.
